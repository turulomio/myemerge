## Configuration

My emerge recomends the use of ccache.

To see statistics with `myemerge --ccache_stats` you must set in /etc/portage/make.conf the following lines:
```bash
  CCACHE_DIR="/var/cache/ccache"
  FEATURES="ccache"
```
